Les filtres

Les filtres permettent d'affiner les données présentées dans vos visualisations. Power BI propose plusieurs niveaux de filtrage :

  • Filtre sur ce visuel : les filtres ne s'appliqueront que sur le visuel sélectionné
  • Filtre dans cette page : les filtres s'appliqueront sur l'ensemble des éléments de la page
  • Filtre dans toutes les pages : les filtres s'appliqueront sur l'ensemble des éléments de toutes les pages

Astuces

  • Un filtre est sélectionné dès lors qu'il passe à une autre couleur que blanc.
  • Pour savoir quels filtres s'appliquent sur un visuel, sélectionnez-le et consultez le volet de filtres.
  • Les filtres dans "Filtres dans toutes les pages" s'appliquent à tous les rapports.

Interactions entre visuels

Les visuels dans Power BI peuvent interagir entre eux automatiquement. Cette fonctionnalité permet une exploration intuitive des données.

Filtrage croisé

  • Cliquer sur un élément d'un graphique filtre automatiquement les autres visuels
  • Maintenir Ctrl pour sélectionner plusieurs éléments
  • Double-clic pour isoler un élément

Survol et détails

  • Survoler un point pour voir les détails
  • Les autres visuels se mettent en surbrillance
  • Clic droit pour plus d'options

Conseil pratique

Utilisez les interactions pour explorer vos données de manière intuitive. Cliquez sur différents éléments pour voir comment ils impactent l'ensemble de votre analyse.

Pourquoi et comment utiliser un signet ?

Si vous reproduisez les mêmes filtres régulièrement, le bon réflexe est de créer un signet.

Les signets permettent de sauvegarder un état spécifique de votre rapport, incluant les filtres appliqués.

  1. Étape 1 - Je sélectionne les filtres souhaités sur mon rapport
  2. Étape 2 - Je crée le signet en utilisant le menu "Affichage" puis "Signets" et "Ajouter"

Avantages des signets

  • Gain de temps considérable
  • Reproductibilité des analyses
  • Partage de vues spécifiques avec collègues

Export et partage

Power BI offre plusieurs options pour exporter et partager vos données et visualisations.

Export vers Excel

  • Clic droit sur un visuel → "Exporter les données"
  • Choisir "Données résumées" ou "Données sous-jacentes"
  • Téléchargement automatique du fichier Excel

Mode plein écran

  • Cliquer sur l'icône "Focus" d'un visuel
  • Utiliser F11 pour le mode plein écran
  • Parfait pour les présentations

Partage sécurisé

Utilisez les fonctionnalités de partage intégrées à Power BI plutôt que les captures d'écran pour maintenir la sécurité des données.

Commentaires collaboratifs

Les commentaires permettent de collaborer directement sur les dashboards et rapports.

  1. Ajouter un commentaire - Cliquez sur l'icône "Commentaire" en haut du rapport
  2. Mentionner un collègue - Utilisez @ suivi du nom pour notifier quelqu'un
  3. Répondre - Cliquez sur "Répondre" pour continuer une discussion
  4. Résoudre - Marquez les commentaires comme résolus une fois traités

Bonnes pratiques

  • Soyez précis dans vos commentaires
  • Mentionnez les personnes concernées
  • Résolvez les commentaires traités

Alertes de données

Configurez des alertes pour être notifié automatiquement lorsque vos données atteignent certains seuils.

Prérequis

Les alertes ne sont disponibles que sur les vignettes de dashboard avec des jauges, KPI et cartes.

  1. Sélectionner la vignette - Cliquez sur la vignette concernée
  2. Accéder aux alertes - Menu "..." → "Gérer les alertes"
  3. Configurer le seuil - Définissez la valeur qui déclenche l'alerte
  4. Choisir la fréquence - Hourly, Daily, ou lors du changement

Cas d'usage

  • Suivi des volumes de souscription / installation
  • Monitoring des KPI importants
  • Détection d'anomalies

Comment créer son reporting en PPT

Vous pouvez intégrer facilement vos rapports Power BI dans PowerPoint pour vos présentations.

Prérequis

Il est nécessaire d'installer un complément dans PowerPoint pour intégrer vos visualisations Power BI.

  1. Ouvrez PowerPoint et allez dans l'onglet "Insertion"
  2. Sélectionnez "Mes compléments" ou "Obtenir des compléments"
  3. Recherchez "Power BI" et installez le complément officiel
  4. Utilisez le complément pour insérer vos rapports Power BI dans vos diapositives

S'abonner aux dashboards

Il est possible de recevoir par mail les dashboards après la mise à jour de manière automatique.

Attention

L'abonnement n'est disponible que pour les collaborateurs avec licence Power BI.

  1. Ouvrez le dashboard auquel vous souhaitez vous abonner
  2. Cliquez sur les trois points "..." en haut à droite
  3. Sélectionnez "S'abonner" ou "Abonnement"
  4. Configurez la fréquence et les détails de votre abonnement
